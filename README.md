# Emacs Docker Image

[![MIT License](http://img.shields.io/badge/license-MIT-blue.svg?style=flat)](LICENSE)

Docker Image for [GNU Emacs](https://www.gnu.org/software/emacs/).

## SUPPORTED TAGS

- `latest`
  - Emacs 24.5.1

## HOW TO USE

```bash
$ docker run \
    --rm \
    --it \
    quay.io/dtan4/emacs \
    emacs -nw
```
